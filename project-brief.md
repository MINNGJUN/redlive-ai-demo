# RedLive AI 小红书直播间 AI 互动体验 Demo

## 项目背景
小红书直播场景中，用户在观看直播时经常会遇到信息密度高、优惠规则复杂、评论滚动过快、问题无法及时被主播回应等体验问题。尤其在商品种草和购买决策过程中，用户需要更快理解直播内容、商品卖点和优惠信息。

## 项目目标
通过 AI 能力增强直播间互动体验，帮助用户快速理解直播内容、总结优惠信息、生成提问内容，并辅助用户完成商品决策，从而提升直播间停留、互动率和转化效率。

## 目标用户
小红书直播间普通观看用户，尤其是边看边决策、关注商品细节和优惠信息的用户。

## 核心体验
用户进入直播间后，可以通过 AI 直播助手快速完成以下操作：
1. 总结主播刚刚讲过的内容
2. 提炼商品优惠信息
3. 根据用户问题生成提问
4. 帮助用户理解商品是否适合自己
5. 快速查看直播间商品卡片和互动入口

## Demo 范围
本 Demo 为前端可交互原型，使用模拟数据实现完整体验闭环，不接真实后端，不接真实直播流。

## 技术方案
- React
- Tailwind CSS
- Vite
- 模拟数据
- Codex 辅助生成代码
- Vercel 部署

## 展示重点
1. 我如何从直播用户体验问题出发定义 AI 机会点
2. 我如何用 Vibe Coding 将需求快速转成可运行 Demo
3. 我如何用 Codex 完成前端搭建、组件拆分和交互实现
4. 我如何作为设计师把控体验、视觉和交互细节
